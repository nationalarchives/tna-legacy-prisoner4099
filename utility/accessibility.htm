Find out more about accessibility on the web

An excellent and comprehensive resource for accessibility on the web has been created by the BBC. We recommend visiting their site if you want to learn more about how to get the most out of your browser on this, and other web sites.

  • Visit the BBC Accessibility web site

Before you visit you might find it useful to see which browser version you are using. Just follow these steps:

  1. Click the option labelled: "Help" in the menu at the top of the window. This is normally the last one in the list.
  2. A menu will pop-up on the screen. The last option should be labelled "About" followed by the name of the program.
  3. Click that option. A small window will then pop-up displaying details of the program you are using.
  4. Once you have found the details you need, you can close that window.
